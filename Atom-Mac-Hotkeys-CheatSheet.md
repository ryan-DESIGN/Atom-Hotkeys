#### LEGEND

`⌘` : Command/Window
`⌃` : Control
`⌥` : Option/Alt
`⇧` : Shift
`⌫` : Delete
`←` : Left Arrow
`→` : Right Arrow
`↑` : Up Arrow
`↓` : Down Arrow
`↩` : Return/Enter
